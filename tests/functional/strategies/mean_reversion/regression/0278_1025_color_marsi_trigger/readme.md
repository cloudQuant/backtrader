# 1025 Exp_ColorMaRsi-Trigger

## 策略概述

该示例是对 MT5 EA `1025_Exp_ColorMaRsi-Trigger` 的 Backtrader 迁移版本。
EA 在 `H4` 周期上组合快慢 `MA` 与快慢 `RSI` 的相对位置，形成 `-1/0/1` 颜色信号，并在信号颜色切换时执行开平仓。

## 原始信号逻辑

1. 计算快慢 `RSI`
2. 计算快慢 `MA`
3. 若快 `MA` 高于慢 `MA`，信号加 `+1`；低于则减 `1`
4. 若快 `RSI` 高于慢 `RSI`，信号再加 `+1`；低于则减 `1`
5. 最终信号被裁剪到 `[-1, 1]`
6. EA 在柱线收盘时读取最近已完成信号柱：
   - 上一信号 `> 0` 且当前信号 `<= 0` 触发买入
   - 上一信号 `< 0` 且当前信号 `>= 0` 触发卖出

## 指标迁移说明

- 指标仅依赖内置 `iRSI` 与 `iMA`，可在 Python 中直接重建
- 保留默认 `H4` 信号周期与固定 `SL/TP`
- 保留源码的颜色信号离散化逻辑

## 数据与运行

- 基础数据：`../../../datas/XAUUSD_M15.csv`
- 信号周期：`H4`
- 运行：`python run.py`
- 绘图：`python run.py --plot`
