# 0520 布林带 RSI

## 策略概述

该策略是对 MT5 EA `0520_布林带_RSI` 的 Backtrader 迁移版本。

- 使用三组不同标准差的布林带构造分层入场/出场价格
- 价格触及上轨区域时做空，触及下轨区域时做多
- 可选 RSI 与 Stochastic 过滤
- 支持多种离场带位定义

## 核心逻辑

1. 根据参数计算三组布林带：标准差、半标准差、双标准差。
2. 依据 `enter` 参数，从不同带位之间生成做空/做多触发价。
3. 当价格进入上方触发区时寻找做空；进入下方触发区时寻找做多。
4. `RSI` 与 `Stochastic` 过滤器启用时，必须同时满足阈值条件。
5. 持仓按 `closure` 指定的带位、固定 `SL/TP` 进行管理。

## 迁移说明

- 原版 `OnlyOnePosition=false` 时允许更密集重复开仓；迁移版默认保留 `OnlyOnePosition=true` 这一主路径。
- 迁移版保留原 EA 的均值回归方向：上方区域偏空、下方区域偏多。

## 运行方式

```bash
python run.py
```
