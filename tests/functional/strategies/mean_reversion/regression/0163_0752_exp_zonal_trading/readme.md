# 0752 Exp_Zonal_Trading

## 策略概述

该策略是对 MT5 EA `0752_Exp_Zonal_Trading` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 使用 `AO` 与 `AC` 两个 Bill Williams 指标的颜色状态
- 两者同为绿色时做多
- 两者同为红色时做空
- 任一指标变色时平仓

## 核心逻辑

1. 在高周期上分别计算 `AO` 与 `AC`。
2. 用柱体相对前一根的增减近似原指标的颜色状态。
3. 当 `AO` 与 `AC` 同时为多头色时开多；同时为空头色时开空。
4. 如果已有多单且任一状态转为空头，则平多；空单则对称处理。
5. 新仓使用固定 `SL/TP`。

## 主要参数

- `ao_tf_minutes`
- `ac_tf_minutes`
- `stop_loss`
- `take_profit`

## 对齐说明

- 原 EA 直接读取 `iAO` 与 `iAC` 的颜色缓冲区 `buffer 1`；当前版本通过 AO/AC 柱值相对前一柱的变化方向重建颜色态。
- 原代码把 `0` 解释为多头色、`1` 解释为空头色；迁移版本保持这一语义。
- 原 EA 中的 `AO_HTF` 与 `AC_HTF` 只用于可视化，交易本体不依赖它们的额外资源文件。
