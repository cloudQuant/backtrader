# 0352 Exp_BlauTVI_Tm

## 策略来源

- MT5 源码：`ea/0352_Exp_BlauTVI_Tm/Exp_BlauTVI_Tm.mq5`
- 指标源码：`ea/0352_Exp_BlauTVI_Tm/BlauTVI.mq5`
- 当前实现：`examples/0352_exp_blautvi_tm/`

## 策略逻辑

- 基于 `BlauTVI` 直方图方向变化交易。
- 当前近似使用三层 `EMA` 平滑后的 `TVI` 值：
  - 若上一根值低于再前一根，而当前值重新高于上一根，则触发开多，同时允许平空。
  - 若上一根值高于再前一根，而当前值重新低于上一根，则触发开空，同时允许平多。
- 保留 `TimeTrade` 交易时段控制：超出设定时段时，平掉现有仓位且不再开新仓。
- 保留固定 `SL/TP` 主流程。

## 与源码一致/差异说明

- 当前版本按源码结构保留了 `BlauTVI` 方向反转开平仓与交易时段控制主流程。
- `TradeAlgorithms` 库文件未直接收录在当前目录中；当前迁移按同仓库内同类模板的单品种单仓语义处理开仓。
- 原指标内部使用多层 `XMA` 平滑和成交量类型选项；当前版本先用 tick volume + `EMA` 链式近似实现默认参数流程。
- 原文示例为 `USDJPY H1`；当前仓库没有对应数据，因此这里使用 `XAUUSD_M15` 重采样到 `H4` 做可运行验证。
