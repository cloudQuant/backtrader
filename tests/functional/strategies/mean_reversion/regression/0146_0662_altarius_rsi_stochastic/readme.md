# 0662 Altarius RSI 随机震荡

## 策略概述

该策略是对 MT5 EA `0662_Altarius_RSI_随机震荡` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 两套 `Stochastic`
- 一套 `RSI`
- 无固定 `SL/TP`，主要依靠条件平仓
- 按连续亏损数量衰减开仓手数

## 核心逻辑

1. 开多条件：
   - `Stochastic(15,8,8)` 主线高于信号线
   - 主线低于 `50`
   - `Stochastic(10,3,3)` 主线与信号线差值大于 `5`
2. 开空条件与之对称：
   - `Stochastic(15,8,8)` 主线低于信号线
   - 主线高于 `55`
   - `Stochastic(10,3,3)` 主线与信号线差值大于 `5`
3. 平多条件：
   - `RSI > 60`
   - `Stochastic(15,8,8)` 信号线开始下拐
   - 信号线位于 `70` 上方
4. 平空条件与之对称。
5. 手数按账户资金和连续亏损数量做衰减近似。

## 迁移说明

- 原 EA 要求对冲账户，但实际开平仓流程是单仓风格；迁移版按单净仓实现。
- 原版 `PeriodRSI` 输入参数未实际用于 `iRSI` 句柄，源码固定使用周期 `3`；迁移版保持与源码一致的指标周期。
- 示例使用 `XAUUSD_M15.csv` 并按 `H1` 压缩运行，以适配当前工作区可用数据文件。

## 主要参数

- `lots`
- `maximum_risk`
- `decrease_factor`
- `period_rsi`

## 运行方式

```bash
python run.py
```
