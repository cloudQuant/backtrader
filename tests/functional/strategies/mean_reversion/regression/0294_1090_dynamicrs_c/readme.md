# 1090 Exp_DynamicRS_C

## 策略概述

该示例是 MT5 EA `1090_Exp_DynamicRS_C` 的 Backtrader 迁移版本。
EA 在 `H4` 周期上读取 `DynamicRS_C` 的颜色状态缓冲区，并在指标方向切换时执行开平仓。

## 原始信号逻辑

EA 只读取指标的颜色缓冲区 `buffer1`：

- 若上一根颜色不为 `0` 且当前颜色切换为 `0`，则开多并平空
- 若上一根颜色不为 `2` 且当前颜色切换为 `2`，则开空并平多

## 指标迁移说明

`DynamicRS_C` 的核心逻辑是：

- 若当前高点同时低于上一根高点、`Length` 根前高点和前一根指标值，则把指标值更新到当前高点附近
- 若当前低点同时高于上一根低点、`Length` 根前低点和前一根指标值，则把指标值更新到当前低点附近
- 否则指标值延续前值，并根据前两根颜色状态恢复方向色值

## 主要参数

- `length`
- `signal_bar`
- `stop_loss`
- `take_profit`
