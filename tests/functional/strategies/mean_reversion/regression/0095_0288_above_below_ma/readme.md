# 0288 均线上下

## 策略来源

- MT5 源码：`ea/0288_均线上下/above_below_ma.mq5`

## 策略逻辑

- 使用 6 周期 MA。
- 买入信号：`open[0] < ma[0]` 且当前价格低于 `ma[0]`，并且 `ma[1] < ma[0]`。
- 卖出信号：`open[0] > ma[0]` 且当前价格高于 `ma[0]`，并且 `ma[1] > ma[0]`。
- 出现相反信号时，先平反向仓再反手开仓。
- 当前 backtrader 实现使用固定手数近似验证。

## 与源码一致/差异说明

- 保留了价格相对 MA 位置与 MA 当前/前一柱方向联合判定的主流程。
- 原 EA 同时支持固定手数与风险百分比；当前实现先使用固定手数近似验证可运行性。

## 运行方式

```bash
python run.py
```

## 文件说明

- `strategy_above_below_ma.py`：策略实现。
- `run.py`：读取 `XAUUSD_M15.csv` 后回测。
- `config.yaml`：策略参数、数据区间和回测配置。
