# 0515 KDJ 智能交易系统

## 策略概述

该策略是对 MT5 EA `0515_KDJ_智能交易系统` 的 Backtrader 迁移版本。

- 依据 `KDJ` 指标中的 `KDC = K - D` 与 `K` 变化生成方向信号
- 固定手数
- 固定止损与止盈

## 核心逻辑

1. 当 `KDC` 从负转正，或 `KDC > 0` 且 `K` 继续上升时，做多。
2. 当 `KDC` 从正转负，或 `KDC < 0` 且 `K` 继续下降时，做空。
3. 同时仅保留一笔净头寸。
4. 持仓使用固定 `SL/TP` 管理。

## 迁移说明

- 我没有沿用原 EA 的 `iCustom("KDJ Indicator")` 路径，而是根据同目录 `KDJ Indicator` 源码直接重写了 `KDJ` 计算，避免编译指标依赖。
- 运行上使用 `M15` 源数据重采样到 `H1`，贴近原文中建议的高周期使用方式。

## 运行方式

```bash
python run.py
```
