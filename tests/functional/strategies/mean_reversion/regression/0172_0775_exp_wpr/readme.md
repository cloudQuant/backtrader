# 0775 Exp_WPR

## 策略概述

该策略是对 MT5 EA `0775_Exp_WPR` 的 Backtrader 迁移版本。
当前版本保留了原 EA 的核心结构：

- 在高一级信号周期上计算 `WPR`
- 监控 WPR 与超买/超卖水平的交叉
- 支持 `DIRECT` 与 `AGAINST` 两种方向解释
- 支持开多/开空/平多/平空开关
- 使用固定 `StopLoss / TakeProfit`

## 核心逻辑

1. 从 `M15` 数据重采样出 `H4` 信号周期
2. 在 `H4` 上计算 `WPR(14)`
3. 当 WPR 从高于超卖线下穿到低于等于超卖线时生成买入或反向卖出信号
4. 当 WPR 从低于超买线上穿到高于等于超买线时生成卖出或反向买入信号
5. 根据信号执行开平仓，并用固定 `SL/TP` 管理仓位

## 主要参数

- `trend`
- `wpr_period`
- `high_level`
- `low_level`
- `signal_bar`
- `buy_pos_open`
- `sell_pos_open`
- `buy_pos_close`
- `sell_pos_close`
- `stop_loss`
- `take_profit`

## 对齐说明

- 原 EA 默认在 `H4 WPR` 上生成信号，当前版本保留该设计
- 原码使用 `TradeAlgorithms.mqh` 执行开平仓；当前版本以 Backtrader 的标准订单接口实现等价行为
- 当前采用固定下单量 `0.1` 手来近似默认 `MMMode=LOT` 的最直接语义
- 信号方向以 MQL 源码为准：`DIRECT` 模式下，跌入超卖触发买入，升入超买触发卖出

## 运行方式

```bash
python run.py
```
