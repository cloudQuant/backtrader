# 0741 Exp_Fractal_ADX_Cloud

## 策略概述

该策略是对 MT5 EA `0741_Exp_Fractal_ADX_Cloud` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 依据 `DI+` 与 `DI-` 的交叉做交易
- 指标运行在独立高周期
- 红绿线交叉时在 bar 收盘给出开平仓信号
- 交易使用固定 `SL/TP`

## 核心逻辑

1. 在高周期上构造 `DI+ / DI-` 序列。
2. 若前一根 `DI+ > DI-`，且当前变为 `DI+ <= DI-`，则视为多头触发条件并关闭空头。
3. 若前一根 `DI+ < DI-`，且当前变为 `DI+ >= DI-`，则视为空头触发条件并关闭多头。
4. 若当前持仓与新信号相反，则先平仓后再等待下一次开仓。
5. 每次新仓附带固定 `SL/TP`。

## 主要参数

- `signal_bar`
- `e_period`
- `normal_speed`
- `stop_loss`
- `take_profit`

## 对齐说明

- 原 EA 的交易条件只使用 `Fractal_ADX_Cloud` 的 `DI+ / DI-` 交叉，当前迁移保留这一执行结构。
- 原始指标带有“分形调速”细节；当前迁移用可重建的 `DI+ / DI-` 云近似，以保证示例可运行。
- 原说明也指出 `HTF` 版本仅用于可视化，交易本体不依赖额外图层。
