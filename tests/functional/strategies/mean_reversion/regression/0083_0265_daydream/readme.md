# 0265 Daydream

## 策略来源

- MT5 源码：`ea/0265_Daydream/daydream.mq5`

## 策略逻辑

- 从最近 `Channel` 根已完成 K 线中寻找最高价与最低价。
- 当当前收盘价低于通道最低价时开多。
- 当当前收盘价高于通道最高价时开空。
- 反向信号先平掉反向仓位，再按新方向入场。
- 止盈为虚拟 `TP`，达到目标盈利后按市价平仓。
- 当前 backtrader 实现直接使用 `M15` 数据回测。

## 与源码一致/差异说明

- 保留了通道突破反手与虚拟止盈主流程。
- 源码不设置真实 `SL/TP`，当前 backtrader 版本同样使用市价入场并在运行中检测虚拟止盈条件。
- 原说明未给定仓库内可直接复现实盘品种，因此这里使用 `XAUUSD_M15.csv` 做可运行验证。

## 运行方式

```bash
python run.py
```

## 文件说明

- `strategy_daydream.py`：策略实现。
- `run.py`：读取 `XAUUSD_M15.csv` 后回测。
- `config.yaml`：策略参数、数据区间和回测配置。
