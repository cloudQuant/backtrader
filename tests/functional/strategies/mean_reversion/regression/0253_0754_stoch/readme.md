# 0754 Stoch

## 策略概述

该策略是对 MT5 EA `0754_Stoch` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 以前一根 K 线的高低收计算上下两个限价触发位
- 每天仅布置一组 `BuyLimit` 与 `SellLimit`
- 任一挂单触发后取消另一侧挂单
- 在 `23:59` 统一清空剩余挂单与持仓

## 核心逻辑

1. 取前一根 bar 的 `high/low/close`。
2. 计算
   - `H4 = close + ((high-low)*1.1)/2`
   - `L4 = close - ((high-low)*1.1)/2`
3. 建立 `SellLimit@H4` 与 `BuyLimit@L4`，并分别配置固定 `SL/TP`。
4. 任一侧价格触达后，转为市价持仓，并删除另一侧待触发单。
5. 若到 `23:59` 仍有持仓或待触发挂单，则全部关闭/删除。

## 主要参数

- `take_profit`
- `stop_loss`
- `lots`

## 对齐说明

- 原 EA 用全局变量避免同一天重复挂单；当前迁移使用策略内部的 `last_day` 与挂单状态完成同样约束。
- 原 EA 的挂单在 MT5 中是真实 `BuyLimit/SellLimit`；Backtrader 迁移版本用“价格触发 + 虚拟挂单状态”近似。
- 源码说明建议在 `H1` 使用，当前脚手架仍沿用项目统一的 `XAUUSD_M15.csv` 作为测试数据源。
