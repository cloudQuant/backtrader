# 0955 i-GAP

## 策略概述

该示例是 MT5 EA `Exp_i-GAP` 的 Backtrader 迁移版本。

原 EA 基于 `i-GAP` 指标上下箭头做反转开平仓，并配合固定 `SL/TP` 管理持仓。

## 指标重建

- 指标源码完整，不依赖缺失外部平滑库
- 使用前一根收盘价相对当前开盘价的缺口阈值判断多空箭头
- 箭头点位按源码使用 `ATR * 3 / 8` 偏移
- 默认使用 `H1` 信号周期

## 交易逻辑

- 出现向上箭头时做多并平空
- 出现向下箭头时做空并平多
- 使用固定 `SL/TP`

## 文件

- `strategy_i_gap.py` - 数据加载、指标重建与策略实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
