# 0334 Auto ADX

## 来源

- EA 源码：`ea/0334_自动_ADX/auto_adx.mq5`
- 当前实现：`examples/0334_auto_adx/`
- 回测数据：`examples/../../../datas/XAUUSD_M15.csv`，运行时重采样为 `H1`

## 策略逻辑

- 使用 `ADX` 主线和 `+DI/-DI` 作为方向与强弱过滤。
- 空仓时开仓：
  - `+DI > -DI` 且 `ADX > level` 且 `ADX` 上升时开多。
  - 按源码字面保留卖出条件：`+DI < -DI` 且 `ADX < level` 且 `ADX` 下降时开空。
- 若启用 `Reverse`，持仓期间在反向条件出现时先平仓，等待后续柱线再考虑重开。
- 保留固定 `SL/TP` 与 trailing 管理。

## 与源码一致/差异说明

- 当前版本保留了源码字面的单仓保护与 `Reverse` 平仓逻辑。
- 源码支持 `Lots` 或 `Risk` 二选一；当前实现同时支持固定手数与基于止损距离的风险百分比近似换算。
- 由于现有样本只有 `XAUUSD_M15`，当前回测用 `M15 -> H1` 重采样近似原作者提到的 `H1` 运行环境。
