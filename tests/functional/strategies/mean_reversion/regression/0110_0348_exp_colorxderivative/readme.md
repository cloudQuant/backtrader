## 策略逻辑

- 基于 `ColorXDerivative` 指标的五色状态变化交易。
- 当前近似颜色含义：
  - `0`：零线上方继续走强。
  - `1`：零线上方走弱。
  - `2`：接近零轴。
  - `3`：零线下方回升。
  - `4`：零线下方继续走弱。
- 保留源码中的颜色状态机开平仓规则：
  - `0 -> 非0` 或 `3 -> 4/2` 触发开多。
  - `4 -> 非4` 或 `1 -> 0/2` 触发开空。
  - `0/3` 区域触发空头平仓。
  - `1/4` 区域触发多头平仓。

## 与源码一致/差异说明

- 当前版本按源码结构保留了 `ColorXDerivative` 颜色状态机开平仓主流程。
- `TradeAlgorithms` 库文件未直接收录在当前目录中；当前迁移按同仓库内同类模板的单品种单仓语义处理开仓。
- 原指标对价格差分做 `XMA` 平滑；当前版本使用加权收盘价导数 + `SMA` 近似实现默认参数流程。
- 原文示例为 `GBPUSD H12`；当前仓库没有对应数据，因此这里使用 `XAUUSD_M15` 重采样到 `H12` 做可运行验证。
