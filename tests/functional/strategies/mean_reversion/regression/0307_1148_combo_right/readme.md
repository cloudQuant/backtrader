# 1148 Combo_Right

## 策略概述

该策略是对 MT5 EA `Combo_Right` 的 Backtrader 迁移版本。

原 EA 以 `CCI` 零轴方向作为基础交易系统，再用 3 组线性感知器对卖方向、买方向和总方向进行覆盖判断；不同信号来源对应不同的 `SL/TP` 参数组。

## 交易逻辑

- 基础系统：`CCI > 0` 做多，`CCI < 0` 做空
- `pass=2` 时启用卖方向感知器，若输出为负则覆盖基础信号做空
- `pass=3` 时启用买方向感知器，若输出为正则覆盖基础信号做多
- `pass=4` 时启用总感知器：
  - 总感知器 `> 0` 且买感知器 `> 0` 时做多
  - 总感知器 `<= 0` 且卖感知器 `< 0` 时做空
  - 否则退回基础 `CCI` 信号

## 风控逻辑

- 基础信号使用 `tp1/sl1`
- 卖感知器信号使用 `tp2/sl2`
- 买感知器信号使用 `tp3/sl3`
- 使用固定手数 `lots`

## 文件

- `strategy_combo_right.py` - 数据加载、CCI 基础信号与感知器覆盖实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
