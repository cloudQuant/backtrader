# 0346 Exp_WAMI_Cloud_X2

## 策略来源

- MT5 源码：`ea/0346_Exp_WAMI_Cloud_X2/exp_wami_cloud_x2.mq5`
- 指标源码：`ea/0346_Exp_WAMI_Cloud_X2/wami.mq5`
- 当前实现：`examples/0346_exp_wami_cloud_x2/`

## 策略逻辑

- 使用两个 `WAMI` 层：
  - 慢层主线/信号线位置判断趋势方向。
  - 快层主线/信号线交叉判断入场时机。
- 当慢层 `WAMI > Signal` 时，只接受快层自上向下穿越后的多头触发。
- 当慢层 `WAMI < Signal` 时，只接受快层自下向上穿越后的空头触发。
- 保留慢层方向平仓开关，以及固定 `SL/TP` 主流程。

## 与源码一致/差异说明

- 当前版本按源码结构保留了“慢趋势过滤 + 快层交叉触发”的双时间框架主流程。
- `TradeAlgorithms` 库文件未直接收录在当前目录中；当前迁移按同仓库内同类模板的单品种单仓语义处理开仓。
- 原 `WAMI` 指标为价格一阶差分后串联三层均线与信号均线的动量结构；当前版本使用 `SMA` 链式近似实现默认参数流程。
- 原文示例为 `EURUSD` 上 `H6` 慢层、`M30` 快层；当前使用 `XAUUSD_M15.csv` 重采样近似运行环境。
