# 0753 三个周期的随机振荡

## 策略概述

该策略是对 MT5 EA `0753_三个周期的随机振荡` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 入场由 `M5` 的随机指标交叉触发
- `M15` 与 `M30` 随机指标必须同向确认
- 出场由独立的 `M5 Exit` 随机指标方向决定
- 单笔交易带固定 `SL/TP`

## 核心逻辑

1. 读取 `M5/M15/M30` 三个周期的随机指标。
2. 当 `M5` 当前 `K>D` 且在 `ShiftEntrance` 根之前 `K<D`，并且 `M15/M30` 也都 `K>D`，同时当前收盘高于前收盘时做多。
3. 做空条件完全对称。
4. 若已有多单而 `SignalExit()==2`，则平多；空单则在 `SignalExit()==1` 时平仓。
5. 开仓时使用固定 `SL/TP`。

## 主要参数

- `tp`
- `sl`
- `lots`
- `shift_entrance`

## 对齐说明

- 原 EA 通过四个 `iStochastic` 句柄分别处理三周期入场和一路径离场；当前迁移保留了同样的角色划分。
- 原 EA 用 `lastorder` 避免连续重复同向开仓；当前版本同样保留该约束。
- 项目统一数据源是 `XAUUSD_M15.csv`，因此这里通过重采样生成 `M5/M15/M30` 信号框架。
