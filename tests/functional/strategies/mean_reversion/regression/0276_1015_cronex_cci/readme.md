# 1015 Exp_CronexCCI

## 策略概述

该示例是对 MT5 EA `1015_Exp_CronexCCI` 的 Backtrader 迁移版本。
EA 在 `H4` 周期上计算 `CronexCCI` 指标，并在快慢线交叉时执行开平仓。

## 原始信号逻辑

1. 计算指定价格上的 `CCI`
2. 对 `CCI` 做一次快线平滑
3. 再对快线做一次慢线平滑
4. 在柱线收盘时按源码条件触发：
   - 前一根快线高于慢线，当前快线回落到慢线下方时买入
   - 前一根快线低于慢线，当前快线上穿到慢线上方时卖出

## 指标迁移说明

- 指标核心序列在源码中完整可见，可直接重建为 `CCI + 双平滑`
- 默认参数使用 `MODE_SMA`，因此可用标准均线直接等价表达
- 保留默认 `H4` 信号周期与固定 `SL/TP`

## 数据与运行

- 基础数据：`../../../datas/XAUUSD_M15.csv`
- 信号周期：`H4`
- 运行：`python run.py`
- 绘图：`python run.py --plot`
