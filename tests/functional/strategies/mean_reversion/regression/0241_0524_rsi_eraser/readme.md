# 0524 RSI 橡皮

## 策略概述

该策略是对 MT5 EA `0524_RSI_橡皮` 的 Backtrader 迁移版本。

- 使用 `H1 RSI` 的 `50` 轴作为方向条件
- 用前一日高/低点过滤信号是否有效
- 每个交易日每个方向最多允许一次入场
- 固定止损，止盈为止损的三倍，盈利后移到保本

## 核心逻辑

1. `H1 RSI > 50` 时考虑做多，`H1 RSI < 50` 时考虑做空。
2. 多头要求前一日最低点减去偏移后不超过允许止损范围；空头要求前一日最高点加偏移后不超过允许止损范围。
3. 同一交易日内，同方向只允许一次新开仓。
4. 盈利超过止损距离后，把止损推进到开仓价。

## 迁移说明

- 原版使用 MT5 的 `CMoneyFixedRisk` 做风险仓位；迁移版用账户资金与止损距离做近似风险定仓。
- 原版在 MT5 对冲模式下可更灵活地处理多空独立开仓；迁移版采用单净头寸近似。
- 为保留原逻辑，迁移版在 `run.py` 中把 `M15` 数据重采样出 `H1` 与 `D1` 数据流。

## 运行方式

```bash
python run.py
```
