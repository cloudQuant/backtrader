# 0722 愤怒小鸟 (剥头皮)

## 策略概述

该策略是对 MT5 EA `0722_愤怒小鸟_(剥头皮)` 的 Backtrader 迁移版本。
当前实现聚焦其默认核心路径：

- 用 `RSI` 与最近 K 线方向触发首单
- 用 `CCI_Drop` 触发反向强平
- 按动态 `PipStep` 做同向加仓
- 按整篮子平均价统一设置目标位与保护位

## 核心逻辑

1. 统计最近 `Glubina` 区间波动，得到动态 `PipStep`。
2. 无仓时，根据最近收盘方向与 `RSI` 过滤开首单。
3. 持仓后若价格逆向偏离最近开仓价超过 `PipStep`，则按 `LotExponent` 加仓。
4. 整体仓位按平均价统一推导 `TP/SL`。
5. 若 `CCI` 反向超阈值，则整组强平。

## 迁移范围

该示例覆盖原 EA 的主要篮子逻辑，但没有逐项扩展所有附加选项。重点保留：

- `CCI_Drop`
- 动态步长
- 同向加仓
- 平均价目标位
- equity stop 开关

## 主要参数

- `lot_exponent`
- `default_pips`
- `glubina`
- `take_profit`
- `stoploss`
- `cci_drop`
- `rsi_min`
- `rsi_max`
