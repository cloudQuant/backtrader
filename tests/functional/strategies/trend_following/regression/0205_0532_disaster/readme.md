# 0532 Disaster

## 策略概述

该策略是对 MT5 EA `0532_Disaster` 的 Backtrader 迁移版本。

- 跟踪价格与长周期 `MA(590)` 的距离
- 使用动态更新的 `Buy Stop / Sell Stop` 挂单
- 只管理挂单，不对已触发仓位进行额外移动

## 核心逻辑

1. 当价格高于长周期均线一定距离时，删除反向卖出挂单，并维护向上的买入突破挂单。
2. 当价格低于长周期均线一定距离时，删除反向买入挂单，并维护向下的卖出突破挂单。
3. 挂单价格按 `trailing_step` 动态重挂。
4. 挂单触发后，持仓只按固定 `SL/TP` 管理。

## 迁移说明

- 原版通过 MT5 原生挂单和定时 `timeout` 更新；迁移版用内部状态模拟挂单与超时节奏。
- 原版只追踪挂单、不追踪持仓；迁移版保留这一点。

## 运行方式

```bash
python run.py
```
