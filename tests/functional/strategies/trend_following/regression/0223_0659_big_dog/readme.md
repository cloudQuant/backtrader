# 0659 BIG DOG

## 策略概述

该策略是对 MT5 EA `0659_大狗(BIG_DOG)` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 在固定时间窗内统计高低点
- 于时间窗结束后布置双向突破意图
- 箱体过宽则不交易
- 止损使用对侧水平
- 未触发挂单在次日删除

## 核心逻辑

1. 在 `start_hour` 到 `stop_hour` 之间统计当日价格区间高低点。
2. 仅当高低差小于 `max_point` 时才允许交易。
3. 在窗口结束后：
   - 上方放置 `BuyStop`
   - 下方放置 `SellStop`
4. 若当前价距离对应边界不足 `distance_max_min`，则不放该方向的突破单。
5. 一旦任一方向被触发，另一侧意图取消。
6. 持仓使用：
   - `TP = entry ± take_profit`
   - `SL = 对侧箱体边界`
7. 未触发意图在新交易日删除。

## 迁移说明

- 原 EA 使用 MT5 原生挂单；迁移版在 Backtrader 中以“待触发突破价位”近似重建。
- 原 EA 要求对冲账户，但其交易约束实际是单方向单仓；迁移版按单净仓实现。
- 示例使用当前工作区已有的 `XAUUSD_M15.csv` 数据，因此参数含义保持不变但交易品种已替换为样例数据品种。

## 主要参数

- `start_hour`
- `stop_hour`
- `max_point`
- `take_profit`
- `distance_max_min`
- `lots`

## 运行方式

```bash
python run.py
```
