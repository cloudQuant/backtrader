# 0651 e-TurboFx

## 策略概述

该策略是对 MT5 EA `0651_e-TurboFx` 的 Backtrader 迁移版本。

- 分析最近 N 根 bar 的 K 线形态
- N 根连续熊势 bar 且柱体递增 → 买入信号
- N 根连续牛势 bar 且柱体递增 → 卖出信号
- 固定 SL/TP，单仓

## 核心逻辑

1. 检查前 `depth_analysis` 根 bar 是否全部同为阳线或阴线。
2. 同时检查每根 bar 的柱体是否大于前一根。
3. 若全部为阴线且柱体递增 → 做多（反转预期）。
4. 若全部为阳线且柱体递增 → 做空（反转预期）。
5. 持仓使用固定 `SL/TP` 管理。

## 迁移说明

- 原 EA 只允许单仓位（有持仓即退出）；迁移版保持这一约束。
- 示例使用 `XAUUSD_M15.csv` 并按 `H1` 压缩运行。

## 主要参数

- `depth_analysis`
- `stop_loss`
- `take_profit`
- `lots`

## 运行方式

```bash
python run.py
```
