# 0541 Flat_Channel

## 策略概述

该策略是对 MT5 EA `0541_平缓通道` 的 Backtrader 迁移版本。

- 使用 `StdDev` 递增区间识别平缓通道
- 在通道上下边界放置突破挂单
- `TP = 1 * 通道宽度`，`SL = 2 * 通道宽度`
- 支持 breakeven 与一次亏损后放大手数

## 核心逻辑

1. 根据 `StdDev` 的变化寻找一段平缓区间。
2. 用该区间最高价/最低价定义通道边界。
3. 若通道宽度位于 `canal_min` 与 `canal_max` 之间，则在上沿和下沿放置突破挂单。
4. 价格触发上沿则做多，触发下沿则做空。
5. 持仓后在达到 `fibo_tral` 对应阈值时把止损推到 breakeven。

## 迁移说明

- 原版依赖 MT5 真正的挂单与订单生命周期；迁移版通过策略内部状态模拟挂单与到期失效。
- 原版还包含时间过滤和更细的资金管理分支；迁移版保留核心通道突破、保本和一次亏损后放大量逻辑。

## 运行方式

```bash
python run.py
```
