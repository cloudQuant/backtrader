# 0735 EMA_WMA

## 策略概述

该策略是对 MT5 EA `0735_EMA_WMA` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 使用 `WMA(8)` 与 `EMA(28)`
- 在新 bar 上判定交叉
- 金叉做多，死叉做空
- 若持有反向仓位，则先平旧仓再开新仓
- 固定 `SL/TP`
- 依据资金风险比例动态计算下单手数

## 核心逻辑

1. `EMA` 与 `WMA` 都基于 `open` 价计算。
2. 当 `WMA` 从下向上穿越 `EMA` 时做多。
3. 当 `WMA` 从上向下穿越 `EMA` 时做空。
4. 已有反向持仓时，先平仓，再沿新方向重开。
5. 每次开仓都按固定点数设置止损和止盈。

## 主要参数

- `period_ema`
- `period_wma`
- `risk`
- `stop_loss`
- `take_profit`

## 对齐说明

- 原 EA 使用 `PRICE_OPEN` 计算均线，迁移版保持一致。
- 原版手数来自自由保证金与 `risk` 百分比，迁移版按同样思想用 `cash / margin_per_lot` 近似重建。
- 原版只在新 bar 检查信号；Backtrader 示例天然按 bar 级别执行。
