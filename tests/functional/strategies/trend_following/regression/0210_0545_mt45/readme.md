# 0545 MT45

## 策略概述

该策略是对 MT5 EA `0545_这是一个跨平台_МТ45_EA_交易。` 的 Backtrader 迁移版本。

- 第一笔固定做多
- 之后按 `买-卖-买-卖` 交替
- 亏损后按 `KL` 放大下一笔手数
- 放大超过 `ML` 后重置为初始手数 `LT`

## 核心逻辑

1. 空仓时在新 bar 开始建立下一笔交易。
2. 第一笔永远做多，之后每笔交易方向交替。
3. 交易使用固定 `SL/TP`。
4. 若上一笔盈利，则下一笔恢复 `LT`；若上一笔亏损，则手数乘以 `KL`，但超过 `ML` 后重置。

## 迁移说明

- 原版通过历史成交结果计算下一笔手数；迁移版在策略内部维护等价状态。
- 这是一个典型的反手加码逻辑，迁移版保留了其核心行为，但未引入 MT4/MT5 平台兼容层细节。

## 运行方式

```bash
python run.py
```
