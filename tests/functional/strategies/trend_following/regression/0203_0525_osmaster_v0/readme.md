# 0525 OsMaSter v0

## 策略概述

该策略是对 MT5 EA `0525_OsMaSter_v0` 的 Backtrader 迁移版本。

- 使用 `OsMA` 最近四根柱的拐点结构判断方向
- 固定手数
- 固定止损与止盈

## 核心逻辑

1. 当 `OsMA` 最近四个值呈现 `高-低-更高-更高` 结构时，做多。
2. 当 `OsMA` 最近四个值呈现 `低-高-更低-更低` 结构时，做空。
3. 同时只保留一笔净头寸。
4. 持仓按固定 `SL/TP` 管理。

## 迁移说明

- 原版核心逻辑非常直接，适合一比一迁移。
- 迁移版把 `iOsMA` 等价为 `MACD Histogram` 近似实现。

## 运行方式

```bash
python run.py
```
