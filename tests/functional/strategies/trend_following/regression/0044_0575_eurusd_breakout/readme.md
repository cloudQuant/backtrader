# 0575 EURUSD_breakout

## 策略概述

该策略是对 MT5 EA `0575_EURUSD_breakout` 的 Backtrader 迁移版本。

- 先计算欧洲时段的窄幅整理区间
- 再在美国时段内寻找区间外突破
- 使用固定 `SL/TP`

## 核心逻辑

1. 到达美国时段开始时，回看前 `24` 根 M15 K 线作为欧洲时段区间。
2. 若该区间宽度不超过 `small_eu_session_pips`，则认定为窄幅整理。
3. 在美国时段内：
   - 前一根 K 线的最低价高于区间上沿 `+3 point` 时做多
   - 前一根 K 线的最高价低于区间下沿 `-3 point` 时做空
4. 每日最多记录一次多头突破和一次空头突破。

## 迁移说明

- 原版运行目标是 `EURUSD M15`；迁移示例继续沿用项目统一样本数据接口。
- 原版允许同日分别记录买/卖突破；迁移版在 Backtrader 单净仓模型下做近似处理，若持有反向仓位则先平后反手。

## 运行方式

```bash
python run.py
```
