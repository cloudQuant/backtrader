# 0596 Pipso

## 策略概述

该策略是对 MT5 EA `0596_Pipso` 的 Backtrader 迁移版本。

- 在指定时段内交易
- 观察最近 `period` 根 K 线形成的最高/最低通道
- 触碰上沿时做空，触碰下沿时做多
- 先平同向旧仓，再按反转逻辑切换方向

## 核心逻辑

1. 计算最近 `period` 根 K 线的最高价和最低价。
2. `close >= high_level` 时，关闭多仓并尝试做空。
3. `close <= low_level` 时，关闭空仓并尝试做多。
4. 止损距离取通道宽度乘以 `SLpp/100 + 1`。

## 迁移说明

- 原版时间窗口支持跨午夜；迁移版保留该语义。
- 原版无止盈；迁移版保留无止盈，仅保留动态计算出的止损价格。

## 运行方式

```bash
python run.py
```
