# 1099 Simple EA

## 策略概述

该示例是对 MT5 EA `1099_简单_EA` 的 Backtrader 迁移版本。
它没有使用外部指标，而是一个最基础的固定止盈止损反手系统。

## 核心逻辑

1. 无持仓时按内部方向状态 `bs` 开仓
2. 初始方向为做多
3. 多单达到 `TP` 后平仓，并继续保持做多方向
4. 多单达到 `SL` 后平仓，并切换为做空方向
5. 空单达到 `TP` 后平仓，并继续保持做空方向
6. 空单达到 `SL` 后平仓，并切换为做多方向
7. 如此循环往复

## 主要参数

- `lot`
- `tp_points`
- `sl_points`
- `point`

## 当前数据与运行方式

- 数据：`../../../datas/XAUUSD_M15.csv`
- 运行：`python run.py`
- 绘图：`python run.py --plot`

## 迁移说明

- 原始 MT5 EA 直接按 tick 检查持仓盈亏点数并决定平仓 / 反手
- 当前版本按 bar 内 `high / low` 触达阈值来近似该逻辑
- 由于原始 EA 没有额外过滤条件，因此该 Backtrader 版本也保持最小状态机实现
