# 0742 PROphet

## 策略概述

该策略是对 MT5 EA `0742_PROphet` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 使用两组独立线性感知器权重
- `daBUY` 与 `daSELL` 可分别开启
- 仅在 `10:00-18:00` 时段允许开仓
- 超过 `18:00` 平仓
- 按移动止损样式推进 `SL`

## 核心逻辑

1. 使用 `Qu(q1,q2,q3,q4)` 对最近几根 K 线高低差做线性组合。
2. 若 `daBUY=true` 且 `Qu(x1..x4) > 0`，在交易时段内开多。
3. 若 `daSELL=true` 且 `Qu(y1..y4) > 0`，在交易时段内开空。
4. 持仓超过 `18:00` 直接平仓。
5. 盘中若价格继续朝盈利方向运动，则按照 `slb/sls` 抬升或下移止损。

## 主要参数

- `da_buy`
- `x1..x4`
- `slb`
- `da_sell`
- `y1..y4`
- `sls`
- `lot`

## 对齐说明

- 原策略要求按周滚动优化权重；当前迁移只复刻执行逻辑，不包含优化流程。
- 原 EA 分别管理买入和卖出的两个感知器；当前实现同样保留双通道结构。
- 原逻辑使用 `M5 EURUSD` 优化说明，但源码本身并未把策略写死在单一品种上。
