# 1153 e-TurboFx

## 策略概述

该策略是对 MT5 EA `e-TurboFx` 的 Backtrader 迁移版本。

原 EA 尝试捕捉强趋势后的回撤：如果最近 `N` 根 K 线在同一方向推进，且实体逐根放大，则按**相反方向**开仓。

## 交易逻辑

- 统计最近 `N` 根 K 线是否全部收阴或全部收阳
- 再检查从第 2 根到第 `N` 根，实体是否逐根大于前一根
- 若最近 `N` 根全部收阴且实体逐根放大，则做多
- 若最近 `N` 根全部收阳且实体逐根放大，则做空
- 无额外主动平仓信号，主要依赖固定 `SL/TP`

## 风控逻辑

- 开仓后设置固定 `StopLoss` 与 `TakeProfit`
- `Lots` 使用固定手数

## 文件

- `strategy_e_turbofx.py` - 数据加载、反向连柱实体信号与风控实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
