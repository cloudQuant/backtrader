# 0533 MAMACD

## 策略概述

该策略是对 MT5 EA `0533_MAMACD` 的 Backtrader 迁移版本。

- 使用两条慢速低价 `LWMA` 与一条快速收盘价 `EMA`
- 快速均线先离开慢速均线区，再反向穿回形成触发条件
- `MACD` 方向作为附加过滤

## 核心逻辑

1. 当快速均线先位于两条慢速均线下方时，记录一个潜在多头状态。
2. 当快速均线重新站上两条慢速均线，并且 `MACD` 为正或正在抬升时，做多。
3. 当快速均线先位于两条慢速均线上方时，记录一个潜在空头状态。
4. 当快速均线重新跌回两条慢速均线下方，并且 `MACD` 为负或正在下行时，做空。

## 迁移说明

- 原版通过 `startb / starts` 维护“先离开、后回穿”的触发状态；迁移版保留该状态机。
- 原版没有复杂仓位管理，适合直接迁移为单净头寸版本。

## 运行方式

```bash
python run.py
```
