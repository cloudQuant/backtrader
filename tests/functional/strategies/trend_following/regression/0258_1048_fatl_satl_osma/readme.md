# 1048 Exp_FatlSatlOsma

## 策略概述

该示例是对 MT5 EA `1048_Exp_FatlSatlOsma` 的 Backtrader 迁移版本。
EA 在 `H12` 周期上读取 `FatlSatlOsma` 单线，并在最近三根已完成柱的斜率拐点出现时进行开平仓。

## 原始信号逻辑

1. 指标输出 `FATL - SATL` 的振荡值
2. 若上一段斜率向下，且最新已完成柱重新向上，则产生做多信号
3. 若上一段斜率向上，且最新已完成柱重新向下，则产生做空信号
4. 反向信号出现时允许平掉已有反向仓位
5. 默认使用固定止损与止盈

## 指标迁移说明

- 按源码重建 `FATL(39)` 与 `SATL(65)` 的固定系数滤波器
- 指标输出为 `(FATL - SATL) / Point`
- 由于原指标使用单价格序列输入，本迁移按默认 `close` 序列重建

## 数据与运行

- 基础数据：`../../../datas/XAUUSD_M15.csv`
- 信号周期：`H12`
- 运行：`python run.py`
- 绘图：`python run.py --plot`
