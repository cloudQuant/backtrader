# 1110 cs2011

## 策略概述

该策略是对 MT5 `cs2011` 的 Backtrader 迁移版本。

原 EA 是 2011 年自动交易大赛冠军程序，核心只使用标准 `MACD`：基于零轴穿越和信号线拐点产生方向信号，并以非常激进的账户权益驱动手数进行净头寸调整。

## 交易逻辑

  - 使用 `MACD(fast=30, slow=500, signal=36)`
  - 当主线跨越零轴时触发方向信号
  - 当信号线在零轴同侧出现拐点时也触发方向信号
  - 新信号到来时，直接把净头寸调整到新的目标手数和方向
  - 默认使用 `H1` 指标周期

## 风控逻辑

  - 默认不设置 `sl_points`
  - 固定 `tp_points`
  - 手数按 `risk * equity / deposit_base` 计算，并限制到最小/最大手数范围内

## 文件

- `strategy_cs2011.py` - 数据加载、`MACD` 信号与动态净头寸实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
