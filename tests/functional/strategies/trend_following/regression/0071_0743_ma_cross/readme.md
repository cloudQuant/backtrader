# 0743 均线交叉

## 策略概述

该策略是对 MT5 EA `0743_均线交叉` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 使用两条不同参数的移动均线
- 上穿时开多，下穿时开空
- 出现反向交叉时平掉当前仓位
- 新仓只在新 bar 触发时建立

## 核心逻辑

1. 分别计算 `MA1(period=3)` 与 `MA2(period=13)`。
2. 若 `MA1` 从下向上穿越 `MA2`，则产生做多信号。
3. 若 `MA1` 从上向下穿越 `MA2`，则产生做空信号。
4. 如果持有多单且出现向下交叉，则平多；空单则在向上交叉时平仓。
5. 单次开仓使用固定手数。

## 主要参数

- `ma_1_period`
- `ma_2_period`
- `ma_1_method`
- `ma_2_method`
- `ma_1_price`
- `ma_2_price`
- `lot`

## 对齐说明

- 原 EA 支持不同均线算法和不同价格源，当前迁移同样保留参数化能力。
- 原实现开仓限制在新 bar 上，当前 Backtrader 版本使用 bar 驱动结构自然匹配该约束。
- 交叉判断按源码的“当前值 vs 另一条均线当前值 + 前一值”形式进行近似复刻。
