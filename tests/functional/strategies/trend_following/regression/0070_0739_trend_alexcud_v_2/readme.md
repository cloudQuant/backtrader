# 0739 TREND_alexcud v_2

## 策略概述

该策略是对 MT5 EA `0739_TREND_alexcud_v_2` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 三个时间框架共同参与投票
- 每层由 5 条 MA 的方向与 1 个 AC 方向共同决定
- 汇总后形成 `Signal = -2/-1/0/1/2`
- 根据信号强度阈值开仓和平仓，并支持 trailing stop

## 核心逻辑

1. 在 `M15 / H1 / H4` 三个时间框架上计算 `MA(5/8/13/21/34)` 的斜率方向。
2. 在 `TF1` 与 `TF3` 上额外计算 `AC` 的方向票。
3. 每个时间框架按 6 票折算为百分比强度，单层满足 `>50` 或 `>=75`。
4. 三层同时一致时，形成 `Signal=±1` 或 `Signal=±2`。
5. `Signal` 超过 `OpenLevel` 时开仓，反向超过 `CloseLevel` 时平仓。

## 主要参数

- `open_level`
- `close_level`
- `ma_periods`
- `stop_loss`
- `take_profit`
- `trailing_stop`

## 对齐说明

- 原 EA 固定使用三层时间框架和五条均线，当前迁移保持同样结构。
- 原 `Signal` 是投票制而不是单一交叉信号，迁移版也保留分层评分机制。
- AC 方向判定按源码的最近 4 根值关系近似重建。
