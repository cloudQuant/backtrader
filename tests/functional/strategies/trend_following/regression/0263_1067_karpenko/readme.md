# 1067 Exp_Karpenko

## 策略概述

该示例是 MT5 EA `1067_Exp_Karpenko` 的 Backtrader 迁移版本。
EA 的入场信号来自 `Karpenko` 主图指标，交易在其两条缓冲线于已完成信号柱发生交叉时触发。

## 原始信号逻辑

EA 读取 `Karpenko` 指标的两条缓冲线：

- `Ind`：`UpBuffer`
- `Sign`：`DnBuffer`

交易条件：

- 若上一根 `Ind > Sign`，且当前 `Ind <= Sign`，则开多并平空
- 若上一根 `Ind < Sign`，且当前 `Ind >= Sign`，则开空并平多

默认参数：

- `InpInd_Timeframe = H4`
- `Basic_MA = 144`
- `History = 500`
- `SignalBar = 1`

## 指标迁移说明

`Karpenko` 的计算步骤为：

- 先对收盘价计算 `Basic_MA` 周期均线作为基线 `base`
- 再对 `History` 周期内的平均波幅 `mean(high-low)` 作为初始距离
- 若当前高点高于基线，则按 `1.618` 递推放大上侧距离，得到 `base + up`
- 若当前低点低于基线，则按 `1.618` 递推放大下侧距离，得到 `base - dw`
- 当价格未上破基线时输出下侧带，否则输出上侧带，另一条线始终为 `base`
