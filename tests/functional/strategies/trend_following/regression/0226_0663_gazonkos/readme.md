# 0663 gazonkos

## 策略概述

该策略是对 MT5 EA `0663_gazonkos` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的“动量后等回滚”状态机结构：

- `STATE 0` 等待允许交易
- `STATE 1` 等待价格动量信号
- `STATE 2` 等待同一小时内的价格回滚
- `STATE 3` 执行开仓

## 核心逻辑

1. 当当前小时未发生过开仓且未超过 `active_trades` 上限时，进入等待信号状态。
2. 若 `close[t2] - close[t1] > delta`，记录多头动量信号。
3. 若 `close[t1] - close[t2] > delta`，记录空头动量信号。
4. 在同一小时内继续跟踪价格：
   - 多头时记录冲高后的最高价，等待回落 `rollback`
   - 空头时记录下探后的最低价，等待反弹 `rollback`
5. 一旦回滚幅度达到阈值，则按方向开仓，并设置固定 `SL/TP`。

## 迁移说明

- 原 EA 要求对冲账户，但主体交易逻辑仍是有限加仓上限约束；迁移版默认按单净仓风格实现，并保留 `active_trades` 参数语义。
- 原 EA 强调应使用真实分时测试；迁移版用当前工作区可用数据做 bar 级近似重建。
- 原版没有复杂外部指标依赖，核心状态机已直接迁移。

## 主要参数

- `take_profit`
- `rollback`
- `stop_loss`
- `t1`
- `t2`
- `delta`
- `active_trades`
