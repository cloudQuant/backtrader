# 1075 MA

## 策略概述

该示例是 MT5 EA `1075_MA` 的 Backtrader 迁移版本。
它基于两条移动均线的方向一致性与相对位置来触发交易，不是最简单双均线瞬时交叉。

## 原始信号逻辑

默认参数为：

- 快线：`LWMA(3)`
- 慢线：`EMA(57)`
- `StopLoss=100`
- `TakeProfit=100`

买入条件同时满足：

- 慢线连续三根上升
- 快线连续三根上升
- 上一根收盘价高于慢线上一根数值
- 当前慢线高于当前快线

卖出条件反向：

- 慢线连续三根下降
- 快线连续三根下降
- 上一根收盘价低于慢线上一根数值
- 当前慢线低于当前快线

## 指标迁移说明

Backtrader 版本直接使用：

- `WeightedMovingAverage(period=3)` 对应原始 `MODE_LWMA`
- `ExponentialMovingAverage(period=57)` 对应原始 `MODE_EMA`

并在每根新 K 线收盘后检查原 EA 的四个多头/空头条件，使用固定止损止盈管理仓位。
