# 0558 Nevalyashka

## 策略概述

该策略是对 MT5 EA `0558_Nevalyashka` 的 Backtrader 迁移版本。

- 启动后先开一笔初始空单
- 每次平仓后都反向再开下一笔
- 每一笔都带固定 `SL/TP`

## 核心逻辑

1. 首次启动后先建立一笔空单。
2. 当前仓位关闭后，下一笔始终按与上一笔相反的方向建立。
3. 每一笔都使用固定 `stop_loss` 与 `take_profit`。

## 迁移说明

- 原版通过读取上一笔持仓方向来决定下一笔方向；迁移版在 Backtrader 中使用内部状态机进行等价近似。

## 运行方式

```bash
python run.py
```
