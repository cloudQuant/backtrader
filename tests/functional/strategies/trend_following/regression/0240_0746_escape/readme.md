# 0746 escape

## 策略概述

该策略是对 MT5 EA `0746_escape` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的主要结构：

- 只在新 bar 评估信号
- 使用 `M5` 上的两条均线 `MA(5)` 与 `MA(4)`
- 无持仓时，前一根 `M5 close < MA(5)` 则做多
- 无持仓时，前一根 `M5 close > MA(4)` 则做空
- 多空分别使用不同的 `SL/TP`

## 核心逻辑

1. 重采样出 `M5` 信号序列。
2. 计算 `MA(5)` 与 `MA(4)`。
3. 当前无持仓时，若上一根 `M5 close` 位于 `MA(5)` 下方则开多。
4. 当前无持仓时，若上一根 `M5 close` 位于 `MA(4)` 上方则开空。
5. 开仓后仅依靠固定 `SL/TP` 管理出场。

## 主要参数

- `take_profit_l`
- `take_profit_s`
- `stop_loss_l`
- `stop_loss_s`
- `lots`

## 对齐说明

- 原 EA 用 `PERIOD_M5` 固定信号周期，当前迁移通过重采样保留这一结构。
- 原源码没有额外的主动平仓逻辑，当前版本同样保持“只靠 `SL/TP` 出场”。
- 原注释强调新 bar 才检查条件以减少 CPU 占用；Backtrader 的 bar 驱动模型天然匹配这一点。
