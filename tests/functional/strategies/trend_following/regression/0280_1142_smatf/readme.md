# 1142 Smatf

## 策略概述

该策略是对 MT5 EA `Smatf` 的 Backtrader 迁移版本。

原 EA 在 3 个时间框架上分别评估 5 条 `SMA` 的方向状态，并在 `TF1` 与 `TF3` 上加入 `AC` 动量评分；每个时间框架形成独立的多空总分，最后据此生成 4 档交易信号。

## 交易逻辑

- 在 `TF1/TF2/TF3` 上分别统计 5 条 SMA 的涨跌方向
- 在 `TF1` 与 `TF3` 上用 `AC` 最近 4 根值的结构给出额外评分
- 每个时间框架的总分为 `(5 条 MA + AC 权重) * 12.5`
- 当 3 个时间框架的多头总分都超过阈值时做多
- 当 3 个时间框架的空头总分都超过阈值时做空
- `OpenLevel` 与 `CloseLevel` 控制开平仓所需的信号等级

## 风控逻辑

- 固定 `SL/TP`
- 可选 `Trailing`
- 固定手数 `lots`

## 文件

- `strategy_smatf.py` - 数据加载、多时间框架 SMA/AC 评分与交易逻辑实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
