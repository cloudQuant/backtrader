# 0702 简单 MACD

## 策略概述

该策略是对 MT5 EA `0702_简单_MACD` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 只在新 bar 决策
- 只比较 `MACD MAIN_LINE(1)` 与 `MAIN_LINE(2)`
- `MAIN_LINE` 上升做多
- `MAIN_LINE` 下降做空
- 反向信号时平仓

## 核心逻辑

1. 读取 `MACD` 主线前两根已完成 bar 的值。
2. 若 `MAIN_LINE(1) > MAIN_LINE(2)`，则做多。
3. 若 `MAIN_LINE(1) < MAIN_LINE(2)`，则做空。
4. 若持仓方向与当前信号相反，则先平仓。

## 迁移说明

- 原 EA 是非常简化的理论验证版本，没有额外过滤和风控。
- 迁移版保持这一极简结构，不额外添加过滤器。
