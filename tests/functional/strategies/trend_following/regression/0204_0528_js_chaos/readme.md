# 0528 JS-Chaos

## 策略概述

该策略是对 MT5 EA `0528_JS-Chaos` 的 Backtrader 迁移版本。

- 使用 `AO + Alligator` 判断方向
- 使用最近确认分形作为突破挂单价格
- 同方向最多布置两笔挂单，目标分别对应 `1.618` 与 `4.618`
- 第一笔离场后，将第二笔推进到保本
- 使用 `MA(21) + AC + AO + StdDev` 做 trailing

## 核心逻辑

1. 当 `AO` 向上且为正，同时 `Lips > Teeth > Jaw` 时，构造多头信号。
2. 当 `AO` 向下且为负，同时 `Lips < Teeth < Jaw` 时，构造空头信号。
3. 在趋势方向上，使用最近分形高/低点布置一到两笔突破挂单。
4. 第一笔仓位使用更大手数和较近目标，第二笔仓位目标更远。
5. 第一笔完成后，第二笔触发 breakeven；随后可按 `MA(21)` 继续推进止损。

## 迁移说明

- 原版依赖 MT5 原生挂单、全局变量跟踪第一/第二订单 ticket。迁移版改为内部状态模拟双挂单与分层仓位。
- 原版的 `Signals()` 中 `AC_1/AC_2` 被读取但未实际参与最终方向判定；迁移版保留其 trailing 用途，但方向判定仍以 `AO + Alligator` 为主。
- 原版 `ClosePositions` 与鳄鱼唇线条件相关，迁移版保留这一近似退出逻辑。
