# 0708 Robot_MACD

## 策略概述

该策略是对 MT5 EA `0708_Robot_MACD` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 基于 `MACD` 交叉
- `0` 轴下金叉做多
- `0` 轴上死叉做空
- 反向交叉离场
- 固定 `TP`

## 核心逻辑

1. 当 `MACD` 在零轴下方向上穿信号线时做多。
2. 当 `MACD` 在零轴上方向下穿信号线时做空。
3. 反向交叉出现时立即平仓。
4. 若价格先命中固定止盈，则按 `TP` 离场。

## 迁移说明

- 原 EA 没有显式止损，迁移版也保持这一点。
- 主要复刻其 `MACD` 过滤与反向交叉平仓逻辑。

## 主要参数

- `take_profit`
- `lots`
