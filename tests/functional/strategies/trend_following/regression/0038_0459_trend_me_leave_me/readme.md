# 0459 Trend_Me_Leave_Me

## 策略概述

该策略是对 MT5 EA `0459_Trend_Me_Leave_Me` 的 backtrader 迁移版本。
原 EA 只维护单一仓位，并根据上一笔通过 `SL/TP` 出场的方向与原因，决定下一笔等待买入还是卖出；入场本身再由 `SAR` 与 `ADX<20` 条件过滤。

## 核心逻辑

1. 始终只持有单一净头寸
2. 初始状态先等待买入条件
3. 若上一笔因 `TP` 出场，则下一笔改为等待反向条件
4. 若上一笔因 `SL` 出场，则下一笔继续等待同向条件
5. 入场过滤：`SAR` 位于价格同侧且 `ADX < 20`
6. 支持固定 `SL/TP` 与 breakeven

## 主要参数
