# 1166 MA2CCI

## 策略概述

该策略是对 MT5 EA `1166_MA2CCI` 的 Backtrader 迁移版本。

原 EA 使用双均线交叉作为主方向信号，并要求 `CCI` 同时穿越零轴确认入场。持仓退出依赖反向均线交叉，止损使用上一根完整柱的 `ATR` 值。

## 交易逻辑

- 当快均线上穿慢均线，且 `CCI` 从非正值穿越到正值时做多
- 当快均线下穿慢均线，且 `CCI` 从非负值穿越到负值时做空
- 持有多单时，快均线下穿慢均线则平仓
- 持有空单时，快均线上穿慢均线则平仓
- 开仓时以此前一根完整柱的 `ATR` 距离设置固定止损

## 资金管理

- `lots > 0` 时使用固定手数
- `lots = 0` 时，按 `free_cash * max_risk / 1000` 估算风险手数
- 若连续亏损超过 1 笔，则按 `decrease_factor` 递减新开仓手数

## 文件

- `strategy_ma2cci.py` - 数据加载、信号与风控实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
