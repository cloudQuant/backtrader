# 0657 SimpleTrade

## 策略概述

该策略是对 MT5 EA `0657_SimpleTrade` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 只在新 bar 工作
- 比较 `Open[0]` 与 `Open[3]`
- 无持仓时按方向开仓
- 下一根新 bar 到来时平仓
- 固定 `SL`

## 核心逻辑

1. 若 `Open[0] > Open[3]`，则开多。
2. 否则开空。
3. 若已有仓位，则在下一根新 bar 时直接平仓。
4. 持仓期间若先触发 `SL`，则提前退出。

## 迁移说明

- 原 EA 没有固定 `TP`，核心就是新 bar 节奏下的短持有逻辑，迁移版保持这一结构。
- 原版源码保留了对冲检查的注释但未启用；迁移版按单净仓实现。
- 示例使用 `XAUUSD_M15.csv` 并按 `H1` 压缩运行，以适配当前工作区可用数据文件。
