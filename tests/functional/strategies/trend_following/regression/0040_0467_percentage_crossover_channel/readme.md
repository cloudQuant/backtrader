# 0467 Percentage_Crossover_Channel_EA

## 策略概述

该策略是对 MT5 EA `0467_Percentage_Crossover_Channel_EA` 的 backtrader 迁移版本。
原 EA 有两种模式：
1. 默认通道边界模式：当上一根 K 线低点向下触碰/穿越下边界时做多，当上一根 K 线高点向上触碰/穿越上边界时做空。
2. 中线交叉模式：当收盘价向下穿过中线时做多，当收盘价向上穿过中线时做空。
出现反向信号时先平掉反向仓位，再保持单一净头寸。

## 核心逻辑

1. 使用 `Percent` 参数构建动态中线与上下边界通道
2. 可在“边界触碰”与“中线交叉”两种模式间切换
3. 支持反转信号方向
4. 有反向仓位时先平仓，再按当前信号开仓
5. 支持固定 `SL/TP`

## 主要参数

- `percent`
- `cross_middle`
- `reverse_trade`
- `volume`
- `stop_loss_points`
- `take_profit_points`
