# 1331 Price Cross MA

## 策略概述

该策略是对 MT5 EA `1331_MQL5向导_-_基于价格交叉移动平均线指标的交易信号` 的 backtrader 迁移版本。
当前版本使用 `XAUUSD_M15.csv` 回测，核心逻辑为价格穿越单条移动平均线触发多空信号。

## 核心逻辑

1. 计算单条移动平均线
2. 当价格向上穿越均线时做多
3. 当价格向下穿越均线时做空
4. 出现反向穿越时平仓或反手

## 主要参数

- `ma_period`
- `lot`
- `point`
- `price_digits`

## 当前数据与运行方式

- 数据：`../../../datas/XAUUSD_M15.csv`
- 运行：`python run.py`
- 绘图：`python run.py --plot`

## 对齐说明

- `1329` 是在此基础上增加 ADX 过滤的增强版本
