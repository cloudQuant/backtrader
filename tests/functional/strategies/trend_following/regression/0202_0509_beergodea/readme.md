# 0509 BeerGodEA

## 策略概述

该策略是对 MT5 EA `0509_BeerGodEA` 的 Backtrader 迁移版本。

- 在 K 线开盘后指定分钟检查信号
- 使用 EMA 斜率与当前价格相对位置判断方向
- 开多前先平空，开空前先平多

## 核心逻辑

1. 到达 `time_bar_open` 指定的开盘后分钟数时才评估信号。
2. 若价格低于当前 EMA、当前 EMA 低于上一根 EMA、且价格低于前收，则做多。
3. 若价格高于当前 EMA、当前 EMA 高于上一根 EMA、且价格高于前收，则做空。
4. 新信号到来前先平掉反向仓位。

## 迁移说明

- 原 EA 使用 `CheckVolume` 时以 `2 * lots` 做可用性检查，但实际核心仍是固定手数单次开仓；迁移版保留固定手数主路径。

## 运行方式

```bash
python run.py
```
