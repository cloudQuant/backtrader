# 1077 Exp_SimpleBars

## 策略概述

该示例是 MT5 EA `1077_Exp_SimpleBars` 的 Backtrader 迁移版本。
EA 在高周期上读取 `SimpleBars` 指标颜色状态，并在颜色翻转时执行开平仓。

## 原始信号逻辑

EA 从指标读取颜色缓冲 `buffer4`：

- 若上一根颜色为 `0`，且当前颜色切换为 `1`，则开多并平空
- 若上一根颜色为 `1`，且当前颜色切换为 `0`，则开空并平多

## 指标迁移说明

`SimpleBars` 的核心逻辑是：

- 默认使用 `period=6`
- 若 `useclose=true`，使用收盘价作为突破判断价格
- 若上一状态为多，则当前价格只要高于最近回看区间某根 `low`，维持多色，否则翻为空色
- 若上一状态为空，则当前价格只要低于最近回看区间某根 `high`，维持空色，否则翻为多色
- 指标最终只输出颜色状态，EA 再用颜色切换生成开平仓信号

## 主要参数

- `rperiod`
- `useclose`
- `signal_bar`
