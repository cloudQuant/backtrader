# 0923 3LineBreak

## 策略概述

该示例是 MT5 EA `Exp_3LineBreak` 的 Backtrader 迁移版本。

原 EA 在 `H12` 信号周期上调用 `3LineBreak` 指标，并在指标柱颜色翻转时交易：

- `0` 表示上涨色
- `1` 表示下跌色

## 指标重建

- 使用最近 `Lines_Break=3` 根信号柱的最高价/最低价作为翻转阈值
- 若当前处于上涨状态且当前低点跌破最近三柱最低点，则翻为空头
- 若当前处于下跌状态且当前高点突破最近三柱最高点，则翻为多头
- 保留原指标的颜色索引 `0/1`

## 交易逻辑

- 当趋势颜色由 `1 -> 0` 时做多
- 当趋势颜色由 `0 -> 1` 时做空
- 若允许反向平仓，则在当前趋势持续为对侧颜色时平掉已有仓位
- 保留固定 `SL/TP`

## 文件

- `strategy_3linebreak.py` - 数据加载、指标重建与策略实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
