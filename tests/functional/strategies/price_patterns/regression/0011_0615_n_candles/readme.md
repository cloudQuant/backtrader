# 0615 N Candles

## 策略概述

该策略是对 MT5 EA `0615_N_Сandles` 的 Backtrader 迁移版本。

- 搜索连续 N 根相同方向的 K 线
- 全部阳线则做多，全部阴线则做空
- 无 SL/TP，无仓位管理

## 核心逻辑

1. 检查前 N 根已完成 K 线的方向（`open < close` 为阳线，`open > close` 为阴线）。
2. 若 N 根全为阳线，发出 Buy 信号。
3. 若 N 根全为阴线，发出 Sell 信号。
4. 不做平仓管理，依赖后续信号或手动退出。

## 迁移说明

- 原版仅在新 K 线诞生时检查，迁移版通过 `[-1]..[-N]` 等价实现。
- 原版无 SL/TP，迁移版保留此行为。
- 原版固定手数 `0.01`；迁移版保留对应参数。

## 主要参数

- `n_candles` — 连续同向 K 线数量（默认 3）
- `lots` — 固定手数

## 运行方式

```bash
python run.py
```
