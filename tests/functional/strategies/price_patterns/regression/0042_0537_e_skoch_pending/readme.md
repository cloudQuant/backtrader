# 0537 e-Skoch_Pending

## 策略概述

该策略是对 MT5 EA `0537_e-Skoch_挂单` 的 Backtrader 迁移版本。

- 结合当前周期与 `D1` 的两根历史柱高低点关系设置突破挂单
- 支持独立的买卖 `SL/TP`
- 达到净值收益阈值时平仓

## 核心逻辑

1. 若 `D1` 的 2 号柱高点高于 1 号柱高点，且当前周期 2 号柱高点也高于 1 号柱高点，则更新买入突破挂单。
2. 若 `D1` 的 2 号柱低点低于 1 号柱低点，且当前周期 2 号柱低点也低于 1 号柱低点，则更新卖出突破挂单。
3. 价格触发挂单后进场，并按独立的买卖 `SL/TP` 管理。
4. 总权益达到 `percent_equity` 目标后平仓。

## 迁移说明

- 原版使用 MT5 原生挂单；迁移版用内部状态模拟待触发挂单与重挂逻辑。
- 原版还会删除更差价格的旧挂单；迁移版保留这一核心更新策略。

## 运行方式

```bash
python run.py
```
