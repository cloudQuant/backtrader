# 0716 10 点值 EURUSD

## 策略概述

该策略是对 MT5 EA `0716_10_点值_EURUSD` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 依据前一日高低点放置突破触发位
- 遇到新日时仅建立一组当日突破机会
- 支持 gap 保护
- 当天第一次触发后不再重复下单
- 可选 trailing

## 核心逻辑

1. 读取前一日 `High/Low`。
2. 新交易日开始时建立上破和下破触发位。
3. 价格突破前一日高点上方时做多，跌破前一日低点下方时做空。
4. 首次触发后取消另一侧并停止当天再次建单。

## 主要参数

- `stop_loss`
- `take_profit`
- `use_trailing`
- `trailing_stop_loss`
- `trailing_step`

## 对齐说明

- 原 EA 使用 `BuyStop/SellStop` 挂单；迁移版用 bar 内突破模拟触发。
- 原版在 gap 情况下拒绝挂单；迁移版保留同样的 gap 保护思想。
