# 0588 多头与空头吞噬

## 策略概述

该策略是对 MT5 EA `0588_多头与空头吞噬` 的 Backtrader 迁移版本。

- 检测两根 K 线构成的多头/空头吞噬形态
- 可设置不同形态触发后的方向
- 可在新信号出现时平掉反向仓位

## 核心逻辑

1. **多头吞噬**：后一根阳线完全吞没前一根阴线实体与区间。
2. **空头吞噬**：后一根阴线完全吞没前一根阳线实体与区间。
3. `opposite_signal=true` 时，按原 EA 的方向映射和反向平仓逻辑执行。
4. `opposite_signal=false` 时，多头吞噬默认做多，空头吞噬默认做空。

## 迁移说明

- 原版使用风险百分比动态算手数；迁移版简化为固定手数。
- 源码中的方向映射存在命名与行为反直觉之处；迁移版按源码实际分支保留其行为。

## 运行方式

```bash
python run.py
```
