# 0469 收盘价分形 EA

## 策略概述

该策略是对 MT5 EA `0469_收盘价分形_EA` 的 backtrader 迁移版本。
原 EA 使用 `Fractals at Close prices` 指标，在新 bar 上读取最近两个向下分形和最近两个向上分形：若最近低分形高于前一个低分形，则认为趋势向上并做多；若最近高分形低于前一个高分形，则认为趋势向下并做空。策略带日内交易时间窗，并在出场时使用固定止损、止盈与 trailing stop。

## 核心逻辑

1. 用收盘价构造分形高点与分形低点
2. 比较最近两次低分形，若后者抬高则生成多头方向
3. 比较最近两次高分形，若后者降低则生成空头方向
4. 时间窗外强制平仓，不保留隔离时段持仓
5. 进场后使用固定 `SL/TP` 与 trailing stop 管理仓位
