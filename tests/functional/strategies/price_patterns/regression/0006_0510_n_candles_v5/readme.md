# 0510 N Candles v5

## 策略概述

该策略是对 MT5 EA `0510_N_Candles_v5` 的 Backtrader 迁移版本。

- 识别连续 N 根同向 K 线
- 支持交易时间过滤
- 固定 `SL/TP`
- 支持 trailing
- 支持同方向最多多笔持仓

## 核心逻辑

1. 检查最近 `N` 根已完成 K 线是否全部同向。
2. 若全为阳线则做多，若全为阴线则做空。
3. 仅在允许交易时段内开新仓。
4. 每笔仓位按固定 `SL/TP` 和 trailing 管理。

## 迁移说明

- 原 EA 同时区分对冲账户的最大笔数与净持账户的最大持仓量；迁移版用 tranche 数量和总仓量近似两种限制。

## 运行方式

```bash
python run.py
```
