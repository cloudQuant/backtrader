# 0514 日线级突破点

## 策略概述

该策略是对 MT5 EA `0514_日线级突破点` 的 Backtrader 迁移版本。

- 使用日线开盘价上下的固定突破距离
- 结合上一根 H1 柱实体大小过滤
- 可配置为顺突破方向入场，或反向平仓反手
- 持仓带 trailing

## 核心逻辑

1. 计算前一日 `D1 open` 作为基准。
2. 当当前价格向上/向下偏离日线开盘超过 `break_point`，并且上一根 `H1` 柱体大小落在允许区间内时触发。
3. `close_by_signal=true` 时按原 EA 主路径执行“关反向仓并反手”；否则按突破方向入场。
4. 持仓使用固定 `TP` 与日线突破 trailing 管理。

## 迁移说明

- 迁移版使用 `M15` 数据重采样得到 `H1` 和 `D1`。
- 原 EA 同时支持手工手数与风险仓位；迁移版默认保留手工手数主路径。

## 运行方式

```bash
python run.py
```
