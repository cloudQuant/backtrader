# 0715 MTC 神经网络, 加上 MACD

## 策略概述

该策略是对 MT5 EA `0715_MTC_神经网络,_加上_MACD` 的 Backtrader 迁移版本。
当前实现保留了原 EA 的核心结构：

- 使用固定权重感知机计算方向分数
- `Supervisor()` 根据不同层级组合感知机输出方向
- `MACD` 作为附加方向过滤器
- 两者同向时才开单
- 不同感知机分支切换不同 `SL/TP`

## 核心逻辑

1. 计算三个感知机输出，每个感知机都基于若干历史开盘价差分。
2. `Supervisor()` 根据 `pass_level` 组合感知机结果。
3. `MACD` 给出额外的多空过滤。
4. 当 `MACD` 与感知机方向一致时入场。

## 迁移说明

- 原代码中的“神经网络”并不是外部训练框架，而是固定参数感知机。
- 因此可以在 Backtrader 中直接重建，无需外部模型文件。

## 主要参数

- `x11..x14`
- `x21..x24`
- `x31..x34`
- `p1 / p2 / p3`
- `pass_level`
- `tp1 / sl1 / tp2 / sl2`

## 运行方式

```bash
python run.py
```
