# 0548 根据时间挂单

## 策略概述

该策略是对 MT5 EA `0548_根据时间挂单` 的 Backtrader 迁移版本。

- 在指定时间放置一组买入/卖出突破挂单
- 使用固定止损
- 使用虚拟止盈和指定小时强制平仓

## 核心逻辑

1. 到达 `opening_hour` 时，在当前价格上下按 `distance` 放置一对突破挂单。
2. 价格触发上方挂单则做多，触发下方挂单则做空。
3. 持仓后按虚拟 `take_profit` 管理；到 `closing_hour` 时平仓并删除剩余挂单。

## 迁移说明

- 原版可以在同一小时内重复放置多组挂单；迁移版在单净仓模型下简化为单组待触发挂单。
- 原版 MT5 支持真实挂单；迁移版使用内部状态模拟挂单触发。

## 运行方式

```bash
python run.py
```
