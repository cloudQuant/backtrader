# 0765 最简单的对冲 EA

## 策略概述

该策略是对 MT5 EA `0765_最简单的对冲_EA` 的 Backtrader 迁移版本。
它的结构非常直接：

- 每当出现新 bar 且当前没有持仓时，同时开一笔多单和一笔空单
- 之后只依赖每条腿自己的 `SL/TP` 出场
- 当两条腿都结束后，再开启下一组对冲单

## 核心逻辑

1. 仅在新 bar 上检查是否允许开仓。
2. 若当前无任何持仓，则同时提交一笔 `Buy` 和一笔 `Sell`。
3. 多头腿使用 `stop_loss` 与 `take_profit`。
4. 空头腿按源码中的实际写法，使用 `take_profit` 距离同时设置其上方止损和下方止盈。
5. 任一腿平仓后保留另一腿，直到整组完全结束，再进入下一轮。

## 主要参数

- `lots`
- `stop_loss`
- `take_profit`

## 对齐说明

- 原 EA 依赖 MT5 对冲持仓模式；当前版本在 Backtrader 中通过分层腿账本近似双向同时持仓。
- 原 EA 明确推荐“仅开盘价格”测试方式；当前 bar 级迁移也按新 bar 触发建仓。
- 卖出腿参数按源码 `Sell(... Bid + take_profit*Point(), Bid - take_profit*Point())` 直接保留，没有自行替换为 `stop_loss`。
