# 1141 VLT_TRADER

## 策略概述

该策略是对 MT5 EA `VLT_TRADER` 的 Backtrader 迁移版本。

原 EA 使用 `iVLT` 指标识别“当前柱体振幅跌破前 `period` 根最小振幅”的压缩信号。出现信号后，围绕上一根柱线的高低点同时挂出一组突破单；当其中一边成交后，另一边挂单会被撤销。

## 交易逻辑

- 计算每根柱线的 `high-low` 振幅
- 若上一根柱线振幅小于更早 `period` 根柱线中的最小振幅，且再上一根柱线不满足同样条件，则生成信号
- 在上一根柱线 `high/low` 外侧按 `pending_level` 挂出 `buy stop` 与 `sell stop`
- 一侧成交后撤销另一侧挂单

## 风控逻辑

- 支持固定 `SL/TP`
- 使用固定手数 `lots`
- 单边持仓期间清除残余对冲挂单

## 文件

- `strategy_vlt_trader.py` - 数据加载、VLT 压缩信号与双向突破挂单逻辑实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
