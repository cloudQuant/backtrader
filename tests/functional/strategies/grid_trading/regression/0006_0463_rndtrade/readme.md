# 0463 RndTrade

## 策略概述

该策略是对 MT5 EA `0463_RndTrade` 的 backtrader 迁移版本。
原 EA 通过定时器每隔固定分钟数先平掉已有仓位，再随机决定下一笔是买入还是卖出，并以最小手数开仓。

## 核心逻辑

1. 每隔 `interval_minutes` 触发一次换仓
2. 到达触发时先关闭已有仓位
3. 随机决定下一笔方向：买入或卖出
4. 保持单一净头寸
5. 使用固定随机种子保证回测可重复

## 主要参数

- `interval_minutes`
- `compression_minutes`
- `volume`
- `random_seed`

## 对齐说明

- 原 EA 使用 `EventSetTimer` 驱动；当前版本按 bar 间隔累计分钟数近似复现定时触发
- 原 EA 使用 `MathRand()`；当前版本使用固定种子随机数生成器，以保证回测结果稳定可复现
- 原 EA 使用交易品种的最小手数；当前版本默认以 `0.1` 手回测，便于与现有示例保持一致
