# 0948 Loco

## 策略概述

该示例是 MT5 EA `Exp_Loco` 的 Backtrader 迁移版本。

原 EA 基于 `Loco` 指标颜色切换开平仓，并配合固定 `SL/TP` 管理持仓。

## 指标重建

- 按源码保留 `Loco` 递推逻辑
- 当价格与前值同向时使用 `0.999/1.001` 偏移更新
- 重建颜色索引 `0/1`
- 默认使用 `H8` 信号周期

## 交易逻辑

- 颜色切换到 `0` 时做多并平空
- 颜色切换到 `1` 时做空并平多
- 使用固定 `SL/TP`

## 文件

- `strategy_loco.py` - 数据加载、指标重建与策略实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
