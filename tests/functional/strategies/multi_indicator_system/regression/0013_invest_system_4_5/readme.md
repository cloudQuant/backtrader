# 0457 投资系统 4.5

## 策略来源
- MT5 EA: `ea/0457_投资系统_4.5/invest_system_4.5.mq5`
- Backtrader 实现: `examples/0457_invest_system_4_5/strategy_invest_system_4_5.py`
- 运行脚本: `examples/0457_invest_system_4_5/run.py`

## 核心逻辑
- 使用执行层当前品种的 `M15` 数据，并重采样为 `H4` 信号层。
- 当上一根 `H4` K 线收阳时，在新 `H4` 周期开始后的前 `15` 分钟内尝试做多；收阴时做空。
- 全程只保持单一净头寸，入场时同时挂出固定 `SL/TP`。
- 按账户余额分段提升 `Lots1-4`，并根据上一笔交易盈亏在 `Lot1/Lot2` 之间切换，保留源码中的 `PlanB/LTS/Pribl` 状态机主流程。

## 参数映射
- `InpStopLoss=240`
- `InpTakeProfit=40`
- `Lots1=0.1`
- `Lots2=0.2`
- `Lots3=0.7`
- `Lots4=1.4`
- `M15 -> H4` 重采样近似源码中的执行层/信号层切分

## 回测数据
- 数据：`examples/../../../datas/XAUUSD_M15.csv`（重采样为 `H4` 信号层）
- 区间：`2025-12-03 01:15:00` -> `2026-03-10 09:00:00`
- Bar shift：`15` 分钟

## 对齐说明
- 原 EA 源码硬编码使用 `EURUSD` 的 `H4` K 线方向作为信号，并提示其它品种需要重调参数；当前仓库只提供 `XAUUSD` 样例数据，因此迁移版以当前回测数据品种的 `H4` 重采样替代该信号来源。
- 原 EA 使用 `OnTradeTransaction` 回写 `Pribl`；迁移版在 `notify_trade` 中于平仓时更新同等状态。
- 原 EA 不使用挂单网格、加仓或对冲结构，属于当前单净仓 Backtrader 框架可承载的类型。
