# 0517 Billy 交易系统

## 策略概述

该策略是对 MT5 EA `0517_Billy_交易系统` 的 Backtrader 迁移版本。

- 当前时间帧检测连续走弱 K 线结构
- 其它两个时间帧的 Stochastic 主线都位于信号线上方
- 仅做多
- 允许分批开仓，直到达到最大持仓数

## 核心逻辑

1. 当前时间帧最近四根柱满足逐步走弱的高点与开盘价结构。
2. 两个辅助时间帧的 `Stochastic` 都保持多头状态。
3. 满足条件时开一笔多单。
4. 每笔仓位按固定 `SL/TP` 独立管理。

## 迁移说明

- 原版默认辅助时间帧包含 `M5` 与 `M6`；迁移版直接使用 `M1` 数据源重采样得到 `M15/M5/M6`，保留这种多时段结构。
- Backtrader 为净头寸模型，迁移版通过内部 tranche 状态近似原 EA 的多笔多单管理。

## 运行方式

```bash
python run.py
```
