# 1115 Rock Trader Neuro

## 策略概述

该策略是对 MT5 `Rock-Trader-Neuro` 的 Backtrader 迁移版本。

原 EA 使用最近 7 根柱上的布林带宽度特征，经过归一化后输入一个带固定权重的单神经元，并以 `tanh` 输出的正负号决定多空方向。

## 交易逻辑

- 使用 `Bollinger Bands(20, 2.0)` 计算上轨、中轨、下轨
- 对最近 7 个布林带宽度特征做归一化，并按原始代码填入 14 维输入数组的偶数索引
- 通过固定权重求和并乘以 `2`
- 使用 `tanh` 激活函数得到输出 `out`
- 当 `out < 0` 时做多
- 当 `out > 0` 时做空
- 每根新柱只评估一次，信号反向时反手

## 风控逻辑

- 固定 `stop_loss`
- 固定 `take_profit`
- 固定手数 `lot`

## 文件

- `strategy_rock_trader_neuro.py` - 数据加载、布林带宽度神经元与交易实现
- `run.py` - 回测入口
- `config.yaml` - 参数配置

## 用法

```bash
python run.py
```
